## Bug ID: WA_BUG_001
**Title:** Notification opens wrong chat thread  
**Severity:** High  
**Priority:** High  

**Steps to Reproduce:**
1. Receive a new message notification from Contact A
2. Tap the notification

**Expected Result:**
The app should open the chat thread for Contact A.

**Actual Result:**
The app opens a different chat thread.

**Status:** Open

## Bug ID: WA_BUG_002
**Title:** “Send” button appears active with empty message  
**Severity:** Low  
**Priority:** Low  

**Steps to Reproduce:**
1. Open any chat
2. Leave message input empty
3. Observe the send button state

**Expected Result:**
Send button should be disabled or should not send anything.

**Actual Result:**
Send button appears active (even though no message is sent).

**Status:** Open  
**Linked Test Case:** WA_TC_003
