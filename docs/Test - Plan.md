# Test Plan

## Project Name
WhatsApp Mobile App – Core Features Testing

## Objective
To verify that core WhatsApp features work correctly and deliver a smooth user experience.

## Scope
- Account onboarding (basic)
- Messaging (text)
- Media sharing (image/video)
- Notifications
- Network switching behavior (Wi-Fi ↔ Mobile data)

## Test Type
Manual Testing

## Environment
- Device: Mobile phone
- OS: Android or iOS
- App Version: Current installed version

## Entry Criteria
App installed and user is logged in.

## Exit Criteria
Core test cases executed and defects documented with clear steps and evidence.
