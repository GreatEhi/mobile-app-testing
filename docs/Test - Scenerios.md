# Test Scenarios – WhatsApp Mobile App

## Messaging
1. Send a text message to a saved contact.
2. Receive a text message from a contact.
3. Send a message to an unsaved number (via “click to chat” or new chat).
4. Verify message delivery indicators (sent/delivered/read) where applicable.
5. Verify long message handling (multi-line / long text).

## Media
6. Send an image from gallery.
7. Take a photo in-app and send.
8. Send a short video (≤ 30 seconds).
9. Download/view received media.
10. Verify media send failure handling on poor network.

## Notifications
11. Verify push notification appears for new message.
12. Verify notification opens the correct chat when tapped.
13. Verify muted chat does not trigger notification.

## Network / Reliability
14. Verify message sends on Wi-Fi.
15. Verify message sends on mobile data.
16. Switch Wi-Fi ↔ mobile data during sending and observe behavior.

## UX / Basic Validation
17. Verify “Send” button behavior with empty message field.
18. Verify app does not crash during normal chat usage.
